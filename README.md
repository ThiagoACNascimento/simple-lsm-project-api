### PEQUENO PROJETO DE ESTUDO
Um projeto focado em _back-end_ para criação um site **LMS** simples para estudar os fundamentos e aplicações prática do node.js.

## Objetivo
O projeto teve como objetivo criar um sistema LMS de aulas com diversas funcionalidades, desde as focadas em _usuário_ até _admin_.

## Funcionalidades
O projeto possui várias rotas abrangendo todo o sistema **CRUD**. Rotas como a de autenticação, autorização, criação de aulas, delete de sessões e etc.

## Conclusão
Com um projeto focado em node puro, foi possível observar diversas funcionalidades mais cruas que diversos _Frameworks_ já realizam de forma mais automática. Nesse sentido, ver com essas funções são aplicádas
ajuda no entendimento mais completo de como algumas ferramentas realmente realizão seus trabalhos e como elas impactam dentro do projeto.

## Créditos
O projeto foi desenvolvido seguindos as aulas do curso de node.js do site: https://www.origamid.com
